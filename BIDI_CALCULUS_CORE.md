## 1 В· The carrier: a coherence algebra рқ’һ

A **coherence algebra** is a structure `рқ’һ = (C, вҠһ, вҠҘ, вҹі, |В·|, Оә)`:

- `(C, вҠһ, вҠҘ)` вҖ” a commutative monoid (**superposition**, with void `вҠҘ`).
- `вҹі : рқ•Ӣ вҶ· C` вҖ” a **rotation** action of the circle `рқ•Ӣ = в„қ/2ПҖв„Ө` by вҠһ-automorphisms:
  `вҹівӮҖ = id`, `вҹі_{ПҶ+ПҲ} = вҹі_ПҶ вҲҳ вҹі_ПҲ`, `вҹі_ПҶ(x вҠһ y) = вҹі_ПҶ x вҠһ вҹі_ПҶ y`.
- `|В·| : C вҶ’ в„қвүҘ0` вҖ” an **amplitude** seminorm: `|вҠҘ| = 0`, `|вҹі_ПҶ x| = |x|`, `|x вҠһ y| вүӨ |x| + |y|`.
- `Оә : C вҶ’ [вҲ’1,1]` вҖ” **commitment** (a cosine): `Оә` is вҹі-covariant, `Оә(вҠҘ) = 0`.

**Standard model.** `C = в„қВІ`, `вҠһ = +`, `вҠҘ = 0`, `вҹі_ПҶ =` rotation by `ПҶ`, `|В·| =`
Euclidean norm, `Оә(x) = вҹЁx, eвӮҒвҹ© / |x|`. (A cell value written in polar form `вҹЁa,Оёвҹ©`
is just `aВ·(cos Оё, sin Оё)`.) The calculus is model-independent; this is one model.

**Derived predicates** (fix a deadband `Оҙ вҲҲ (0,1)`, canonically `ВҪ`):

```

                 вҲ’   if Оә(x) < вҲ’Оҙ

`П„` is the runtime symbol; `П°` is the boundary openness (maximal at the crossings
`Оә = 0`). These are *defined*, not primitive.

## 2 В· Terms (the syntax of fields)

A **field** is a term generated by five constructors:

```
F, G  ::=  0                         -- the empty field (inert)
        |  m[ vМ„ ; bМ„ ]               -- a module m: cell-vector vМ„ вҲҲ CвҒҝ, belief bМ„ вҲҲ в„қвҒҝ

        |  m в–·вҹЁw,П„вҹ© m'               -- a channel: weight w вҲҲ в„қ, delay П„ вҲҲ в„қвүҘ0
        |  mвҹҰ F вҹ§                    -- nesting: module m hosts field F  (fractal)

Module names are distinct. A module additionally carries static annotations
(intrinsic frequency `ПүМ„`, prior `bМ„вҒ°`, precision `ПҖ`, action gain) elided in the
grammar. The canonical arity is `n = 6` with read cone `= vМ„[1..3]`, write cone
`= vМ„[4..6]`.

## 3 В· Structural congruence вүЎ

`вүЎ` is the smallest congruence closed under:

```
(вҲҘ-assoc)   (F вҲҘ G) вҲҘ H вүЎ F вҲҘ (G вҲҘ H)

So `(fields, вҲҘ, 0)` is a commutative monoid: composition is order-free, enabling
compositional reasoning. This is the equational backbone.

## 4 В· Reduction (a hybrid relation)

Computation is a binary relation on fields with two faces вҖ” continuous **flow** and
discrete **commit**. A *run* is any interleaving.

### 4.1 Flow `вҹ¶_d`  (continuous, `d вҲҲ в„қвүҘ0`)

`F вҹ¶_d F'` evolves every cell, belief, and plastic weight for real duration `d` under
the field's flow `рқ”Ј` (the afferent-driven vector field: phase turns toward incoming
superposition gated by openness; amplitude relaxes; belief descends prediction error;
plastic weights follow correlation). `рқ”Ј` is globally Lipschitz on bounded amplitude.

### 4.2 Commit `вҹ¶_ОІ`  (discrete, guarded)

Each module may carry a guard `G_m` (a predicate on `F`). The **commit** rule:

```
        G_m fires (zero-crossing)        ГӮ = afferent_m(F)        m' = commit(m, ГӮ)
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ [COMMIT]

where `commit(m, ГӮ)` does, atomically:

1. **snap** each cell a fraction toward its nearest committed pole;
2. **barrier** вҖ” walk the committed trits `rвӮҖ=0, rбөў=r_{iвҲ’1}+П„МӮбөў`; any cell forcing

`commit` is **total**: it either lowers `ОҰ_m` and admissibly latches, or holds. The
event time is located off the realization grid by interpolation, so commits are
event-driven, not clocked.

### 4.3 Cross-scale coupling `bidiОіО”`

For a nested term `mвҹҰFвҹ§`, reduction interleaves parent and child with two coherence
flows: **down** (parent state sets child priors `bМ„вҒ°`) and **up** (child coherence
`Оі(F)` enters the parent's afferent). Their signed difference is the coherence-delta;
distinct nesting paths are distinct reference frames. The same coupling therefore
serves three formal roles: invariant enforcement, reference-frame transfer, and
free-energy exchange.

## 7 В· Metatheorems (executable witnesses)

**T1 вҖ” Preservation.** `F вҹ¶_ОІ F'` вҹ№ every committed module in `F'` is *admissible*
(its trit walk never goes negative). *Proof:* the barrier rotates any rank-violating
cell to its crossing (`П„МӮ=0`), which cannot lower the running rank; induct on cells.
вң” witnessed over 2000 random commits.

**T2 вҖ” Soundness (Lyapunov).** Let `ОҰ(F)=ОЈ_m ОҰ_m`. Then `вҹ¶_ОІ` never increases `ОҰ`
(unconditionally, by the guard's hold-clause); and `вҹ¶_d` never increases `ОҰ` in its
belief component (exact gradient descent), and in its coupling component under
symmetric, delay-free, ungated coupling. Under those stated regimes, `ОҰ` is a
Lyapunov witness for the reduction: runs descend free energy. вң” max `О”ОҰ = 0` over
3000 commits.

**T3 вҖ” Local confluence (diamond).** If modules `m, m'` share no channel and both
guards are enabled, then `вҹ¶_ОІ(m); вҹ¶_ОІ(m') вүЎ вҹ¶_ОІ(m'); вҹ¶_ОІ(m)`. *Proof:* `commit(m)`
reads only `m`'s incident channels and writes only `m`; disjoint footprints commute.
вң” witnessed as `A;B вүЎ B;A`.

**T4 вҖ” Time-determinism & additivity.** `рқ”Ј` is globally Lipschitz, so by
PicardвҖ“LindelГ¶f the flow is the unique solution and `вҹ¶` is a deterministic monoid
action of `(в„қвүҘ0, +)`: `вҹ¶_0 = id_вүЎ` and `вҹ¶_{dвӮҒ}; вҹ¶_{dвӮӮ} = вҹ¶_{dвӮҒ+dвӮӮ}`. вң” witnessed
exactly on grid-aligned durations.

**T5 вҖ” Normal forms / strong normalization.** A committed module is `вҹ¶_ОІ`-irreducible
(a **value**) iff its trit walk is admissible and *localized* (returns to 0). The
saturated such walks number `C_{n/2}` (**Catalan**; 5 for `n=6`); allowing crossings,
`M_n` (**Motzkin**; 51). Under the localizing regime the drain step (proven
terminating) reduces any module to a normal form вҖ” **strong normalization onto a
finite value set**. вң” census `5 / 51`; a localized closure is a stable fixed point.
